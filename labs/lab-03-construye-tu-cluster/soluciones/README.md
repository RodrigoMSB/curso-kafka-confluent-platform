# Soluciones del Lab 03 — Construye tu propio clúster KRaft

Esta carpeta contiene las soluciones de referencia del laboratorio.

## Archivos

| Archivo | Contenido |
|---------|-----------|
| `reporte-resuelto.md` | Respuestas modelo a las preguntas del reporte del lab |
| `respuestas-desafio.md` | Solución comentada del desafío de la Parte 5 |
| `docker-compose-1-broker.SOLUCION.yml` | Compose de referencia para el clúster de 1 broker (Parte 2) |
| `docker-compose-3-brokers.SOLUCION.yml` | Compose de referencia para el clúster de 3 brokers (Parte 4) |
| `env.SOLUCION` | Variables de entorno de referencia para los composes anteriores |

## Cómo usar estas soluciones

1. Intenta primero resolver el lab por tu cuenta siguiendo la guía
   en `../guia/`.
2. Si te trabas en una actividad puntual, consulta solo esa sección
   de la solución correspondiente.
3. Al terminar, contrasta tu reporte con `reporte-resuelto.md` para
   verificar tu comprensión.

> El aprendizaje real está en el proceso de resolver. Las soluciones
> son una herramienta de validación, no un atajo.
